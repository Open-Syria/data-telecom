# Pre-Seed Checklist

Before adding a new telecom source:

- Confirm the source is public and usable for factual reference metadata.
- Check that it does not contain personal subscriber data or live network status.
- Decide which canonical file each fact belongs in.
- Create an import manifest.
- Add source registry metadata.
- Preserve source dates and access timestamps.
- Run schema, import, data, fixture, coverage, and release validation.
