# ID Policy

OpenSyria IDs are stable, lowercase, hyphenated IDs beginning with `sy-`.

Patterns:

- country plan: `sy-national-numbering-plan`
- operator: `sy-syriatel`
- fixed area code: `sy-fixed-area-code-11-damascus-rif-dimashq`
- mobile prefix: `sy-mobile-prefix-093-syriatel`
- number range: `sy-mobile-prefix-090-reserved`

Do not rename IDs unless the existing identity is wrong. If a public label
changes but the entity remains the same, keep the ID stable and update the
display name and source references.
